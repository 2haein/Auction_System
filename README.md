# Used-Auction-System

중고 경매 시스템   [ cau sw공학 프로젝트 ]  
    <br/>
    
### 1. 프로젝트 소개

> 보통의 중고거래에서는 판매자가 물건을 업로드할 때 가격을 정해서 올린다. 
> 기존의 시스템에서 보완된 'UA'는 개인 경매 시스템으로 구매자의 편의성을 높여주고, 판매자의 부담 또한 덜어줄 것이다.
